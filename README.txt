# Nombre del Proyecto

Este proyecto es una aplicación web desarrollada con Django que simula una billetera virtual.
Permite a los usuarios ingresar dinero en su cuenta, realizar transacciones con otros usuarios,
tener el control de gastos a traves de un historial de transacciones. 
Ademas podran agregar una foto de perfil, de esta forma facilitaran el reconocimiento de usuarios
a la hora de realizar transacciones.

## Instrucciones para Ejecutar el Proyecto

Sigue estos pasos para configurar y ejecutar el proyecto localmente:

### Requisitos Previos
- Python 3.x instalado
- Django 5.x instalado
- Un entorno virtual configurado (opcional pero recomendado)
- Base de datos (PostgreSQL)

### Instalación
1. Clona el repositorio en tu máquina local:

   git clone https://github.com/FValentino/starWallet.git
   cd <nombre-del-proyecto>


### Funcionalidades 

1. Usuarios no registrados
    a. Iniciar sesion

2. Usuarios registrados
    a. Ver saldo de la cuenta.
    b. Editar datos personales.
    c. Ingresar dinero.
    d. Realizar transferencias a otros usuarios registrados.
    e. Acceder al comprobante de cada transferencias. 
    f. Destacar usuarios activos.
    g. Acceder al historial de transacciones.

3. Usuarios administradores
    a. Visualizar lista de usuarios.
    b. Activar o desactivar cuentas de usuarios.
    c. Ver informacion de los usuarios.
    d. Editar informacion de los usuarios.
    e. Revisar detalles de la cuentas.
    f. Gestionar los motivos de transferencias.
